概要

ガラポンTVの簡易ブラウザです。
デフォルト設定では再生に Flash Player が必要です。
Android 4.0以降に Flash Player をインストールする方法については以下のリンクを参照してください。
  • Android 4.x端末への FlashPlayer のインストール方法

設定

ガラポンID、パスワード

ガラポンのIDとパスワードを設定します。

再生方法

再生方法を4種類から選択できます。それぞれ特徴があります。
※再生時にリストを長押しするといつでも再生方法を切り替えられます。
再生方法詳細
FlashFlash Player を使用します。
プロトコルはRTMPです。
ほとんどの環境で一番画質や滑らかな再生ができます。
ただしガラポンTVのRTMPサーバが止まって使えなくなることがよくあります。
VideoViewAndroid標準の動画再生機能をを使用します。
プロトコルは HTTP Live Streaming です。
端末によってパフォーマンスや安定性にばらつきがあります。
放送中の番組は再生を開始した時点の終了部分までしか再生できません。
Flashで再生できないときはこの再生方法を選ぶと再生できる可能性が高いです。
ポップアップ
(VideoView)		VideoViewと同じですが、再生画面がポップアップ表示されます。
他のアプリを使いながら同時に視聴できます。
ポップアップはドラッグして移動したり、マルチタッチで拡大・縮小ができます。
他のアプリ端末にインストールされている他のアプリで再生します。
プロトコルは HTTP Live Streaming です。
BSPlayer などが使え、パフォーマンスが良い場合があります。

再生画面の操作

  • 分割状態で再生中に再生画面をタップすると全画面に切り替え
  • 左右フリックで「番組詳細」「再生」「字幕」タブを切り替えられます
  • 「再生」タブで画面右側1/3を上下にスワイプすると音量調整
  • 「再生」タブで画面左側1/3を上下にスワイプすると明るさ調整
